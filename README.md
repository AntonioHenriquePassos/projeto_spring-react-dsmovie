# projeto_spring-react-dsmovie

Projeto site que mostra uma listagem de filmes, os quais estão disponíveis para avaliação do usuário mediante informação de email e
uma nota de 0 a 5. As notas são salvas e a pontuação de cada filme é atualizada.

Segue o link para ver o projeto rodando no Netlify conectado com o Heroku:

https://antonio-henrique-dsmovie.netlify.app/
